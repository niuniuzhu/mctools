版本记录

历史更新

正式版1.3.2

修复故障版本判断过于死板

登录页对故障部署地址的检测从完整 URL 改为 host 与 pathname 判断，避免因为协议、路径形式或附加参数差异导致远程与本地表现不一致，同时补齐严重问题提示文案。

正式版1.3

趣味实验室扩展到 9 项功能

在原有幸运方块、生物对决、昵称生成、掉落轮盘和今日挑战之外，新增宝藏坐标盲盒、村民交易吐槽、Boss 出场台词和联机整活惩罚签，并切换到正式版 1.3。

1.3rc3

补充全站视觉插画

为大多数共享页面追加插画背景、角落缩略图和更明显的卡片层次，减少页面过于空白和过于简洁的问题。

1.3rc2

关闭白天选项

设置页关闭白天切换入口，显示模式统一锁定为晚上，旧的白天本地设置也会自动回到夜间模式。

1.3rc1

取消验证码登录

登录流程改回用户名和密码直接登录，移除登录页验证码展示、刷新和提交校验，减少进入阻力。

beta2.3.3

首页主视觉居中

首页顶部主视觉改为单列居中展示，标题、简介和入口按钮统一收拢到中轴，分类区保持不变。

beta2.3.1

设置页开放并接管日志入口

开放独立设置页，账号信息、VIP 购买、昼夜切换和更新日志入口统一迁入；更新日志页移除返回登录按钮，改为返回设置。

beta2.3.0

设置入口与昼夜模式

新增设置承载页的产品结构，准备承接账号信息、VIP 购买与白天 / 晚上显示切换，减少首页信息堆叠。

beta0.1.0

最早原型建立

建立最小服务结构和静态页面骨架，确认工具箱的登录入口、首页入口和后续模块扩展方向。

beta0.2.0

基础登录与注册流程

补上用户名注册、登录交互和基础页面跳转，让工具箱具备最初可进入的账号体系。

beta0.3.0

指令工作台初版

加入常用 Minecraft 指令生成能力，形成工具箱的第一个核心功能面板。

beta0.5.0

分类入口与页面框架扩展

首页开始采用分类入口方式组织功能，后续教程、配方和更多模块可以独立扩展。

beta0.7.0

会员与主题样式雏形

引入会员区分、界面主题切换和更完整的页面视觉结构，为后续多模块统一风格做准备。

beta0.9.0

工具箱体系成形

在正式版前整合登录、分类、基础工具和服务端支持，使整个工具箱具备独立运行条件。

v1.0.0 正式版

基础工具箱完成初版上线

完成登录注册、指令生成、教程入口、基础会员体系、主题风格切换和服务端静态页面支持，形成工具箱基础结构。

beta1.5.1

性能测试与页面检测独立化

将帧率实时检测和我的世界页面检测从教程中拆出，改为独立分类页面，便于单独使用与后续扩展。

beta1.6.0

分类扩展与账号体验优化

统一版本号到 beta1.6.0，修复头像上传无反馈问题，拆分 AI 为独立分类，补充多类工具页面，并扩展常用指令模板。

beta1.7.0

回退实验更新并补充更新日志

撤销“奶茶工具”分类与免验证码登录的实验更新，恢复验证码登录流程，新增独立更新日志页面，并统一站内版本号显示。

beta1.8.0

iOS 液态玻璃效果实装测试

新增独立的液态玻璃实装测试页面，可直接查看玻璃面板、液态浮层、blur 强度切换，并结合当前设备环境给出可用性提示。

beta1.8.1

液态玻璃时装实验室

将单纯的液态玻璃实装测试升级为风格化实验页，加入冰蓝、极光、日落三套视觉预设，并增强高光、层次和展示气氛，同时补强首页和更新日志页的液态玻璃主题细节。

beta1.9.0

液态玻璃全界面实装

将 liquid 主题从首页与日志页扩展到全站主要界面，统一模板区、收藏区、历史区、AI 与执行器面板、标签、输入框和按钮的玻璃质感，并加入更完整的悬浮和扫光反馈。

v1.2.0 正式版

Linux 部署修复与稳定发布

修复 Linux 服务器部署时新增分类与实验页因静态文件白名单遗漏导致的 404 问题，并调整维护模式控制权限与状态解析逻辑，确保维护状态可以稳定关闭，作为正式版 1.2.0 发布。

beta1.10.0

维护权限并入开发者账号

取消独立维护账号与对应注册入口，维护模式的开启、关闭、维护期登录与状态控制统一由开发者账号负责，减少双角色带来的权限分叉与维护成本。

beta1.11.0

取消维护功能并补充版本号控制

移除维护模式相关接口、访问拦截、开发者维护控制面板和登录页维护态展示，同时保留并补强开发者控制台的版本号读取与保存功能，可直接更新当前站点版本显示与接口版本字段。

beta1.12.0

光影下载中心与双档资源包

新增光影下载中心，提供 Java / 基岩双端的中配与高配四套资源包下载，并补上站内预览图，既可走稳定帧率路线，也可走更强视觉表现路线。
