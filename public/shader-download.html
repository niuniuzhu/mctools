推荐搭配模组

Java 版画质与体验补强

加载核心 Iris Shaders

优先用于 Fabric 环境加载光影，和本页 Java 包最稳，调节阴影、体积云和反射时更直观。

帧率优化 Sodium

如果你打算开更高视距或更高配档位，先上 Sodium 把基础帧率拉稳，再去调光影细节。

纹理补全 Continuity

适合搭配木板、砖块和部分连接材质，让资源包边缘更统一，不容易出现拼接断层。

氛围增强 LambDynamicLights

探索洞穴和夜景时手持物动态发光更明显，和本页高配包的夜景氛围很搭。

基岩版建议保持纯净导入，不要叠加超高清 UI 包或多个同类材质。Java 版常见组合：Fabric API + Iris + Sodium，再按需要补 Continuity 与动态光源。
