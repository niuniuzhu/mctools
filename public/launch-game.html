入口 1

网易版 Java 经典版

适合准备进入网易版 Java 经典版的玩家。主按钮会按你桌面快捷方式里记录的协议，优先尝试直接唤起本机启动器。

按桌面快捷方式启动 打开网易版官网
已按桌面快捷方式解析到的 fevergames 协议处理。如果本机 FeverGames 或网易启动环境已正确关联，就会直接进入；否则通常会无响应。

入口 2

网易版基岩互通版

适合准备进入网易版基岩互通版的玩家。主按钮会优先尝试直接打开已安装的游戏或启动环境。

直接启动游戏 查看网易版说明
如果系统已把该协议关联到互通版或兼容环境，会直接尝试启动；如果没关联，通常需要回官网重新安装。

入口 3

国际 Java

这个入口先关闭，不再从这里直接启动国际 Java。

没装就去下载
如果后面要重新开放，再把这里恢复为直接唤起 Minecraft Launcher 即可。

入口 4

国际基岩版

适合 Windows 上已经装好基岩版的设备，通过系统协议尝试直接拉起游戏。

尝试启动基岩版 去国际版下载页
如果系统没有关联 minecraft:// 协议，说明当前机器还没有可用的基岩版安装环境，或者协议被其他应用接管。

入口 5

Minecraft 1.8 Web 启动

直接在浏览器打开 1.8.8 WASM 页面，无需本机启动器协议，适合快速体验或临时游玩。

打开 1.8 Web 启动页 打开 MCJS 主页
该入口是纯网页启动，不依赖系统协议；如果打不开，通常是网络限制或浏览器兼容性问题。

入口 6

自研沙盒 1.20.1 风格

这是工具箱内置的类 Minecraft 沙盒实验页，支持移动、挖掘、放置和本地存档，打开即玩。

进入自研沙盒
该入口属于自研演示，不依赖官方资源。建议使用桌面端浏览器体验更流畅。
