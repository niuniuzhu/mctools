打开快速说明 打开完整安装指南

第二步

📋 查看 Mod 清单与配置

了解推荐的必选、优化、光影和 UI 模组列表，以及启动参数。

打开 Mod 清单（推荐） 打开 JVM 参数
打开 Manifest（可编辑） 打开版本记录

第三步

🔧 下载与放置 Mod 文件

按 Mod 清单从 Modrinth 或 CurseForge 下载，放入 mods 目录。建议先下载"必选 + 性能优化"组（共 5-7 个 mod）。

推荐下载平台：
🌐 Modrinth：https://modrinth.com（筛选 1.20.1 + Fabric）
🌐 CurseForge：https://www.curseforge.com/minecraft/mods（筛选 1.20.1 + Fabric）

第四步

✅ 运行检查脚本验证

使用自动化脚本检查文件完整性、Java 版本、Mod 数量等。

下载检查脚本（Windows PowerShell）
使用方法：
1. 将 check-pack.ps1 放在整合包根目录
2. 右键打开 PowerShell：powershell -ExecutionPolicy Bypass -File check-pack.ps1
3. 脚本会检查文件、Java 版本和 mod 数量

第五步

🚀 配置启动器与启动

选择你使用的启动器（PCL / HMCL / MultiMC），配置 Java、内存和 JVM 参数。

推荐配置：
🔹 Java：Java 17（关键）
🔹 内存：6GB（中等）～ 8GB（推荐）
🔹 JVM 参数：复制 jvm-args.txt 的全部内容
🔹 首次启动后按需调整画质和距离

常见问题排查

❌ 遇到问题？

❓ 卡在加载界面 → 只保留"必选 + 性能优化"组启动，逐个添加其他 mod
❓ 崩溃报告 → 查看 crash-report，删除冲突的 mod 或找兼容版本
❓ 帧数很低 → 增加内存到 8GB、关闭光影、降低渲染距离
❓ Java 错误 → 确认已安装 Java 17（不是 Java 8 或 11）
