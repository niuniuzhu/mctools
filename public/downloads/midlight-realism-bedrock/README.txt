Midlight Realism Bedrock Pack

定位：中配移动端 / Win10 基岩版视觉增强包。

内容方向：
- 偏暖阳与湿润地表氛围
- 强化石材、泥土、木纹和明暗层次表现
- 保持较轻量，尽量适合中配设备导入

建议环境：
- Minecraft Bedrock Edition 1.20+
- Win10 / 安卓高端机 / 新款平板

建议设置：
- 视距：10-14
- 关闭过高图形附加项
- 与其他超高清 UI 包分开使用

安装方式：
1. 下载 mcpack 文件
2. 双击或在文件管理器中选择用 Minecraft 打开
3. 在全局资源或世界资源中启用

说明：
本包为工具箱内置演示下载包，定位为中配视觉体验模板。
