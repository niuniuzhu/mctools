Midlight Release Kit

This folder contains starter templates for public release and PCL-friendly distribution.

Included files:
- modrinth-release-template.txt
- pcl-import-guide.txt
- modpack-manifest-template.json
- mod-list-template.md

Recommended usage:
1. Publish the actual files first: Java zip, Bedrock mcpack, preview images.
2. Fill in the Modrinth template with version, compatibility, and changelog details.
3. Add the PCL import guide to your release page or README.
4. If shipping a Java modpack instance, fill in the manifest template and mod list.

Do not package:
- account data
- logs
- crash reports
- caches
- machine-specific paths
- temporary download files
