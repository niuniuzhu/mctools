Minecraft 1.20.1 私人整合包（模板）

说明
- 这是一个可直接二次编辑的私人整合包模板，不包含第三方 mod 二进制文件。
- 你可以按 mod-list.md 下载对应 mod 放入 mods 目录，再按 manifest 发布。
- 目标版本：Minecraft 1.20.1
- 推荐加载器：Fabric（稳定、轻量、适合优化向整合）

目录建议
- mods/
- config/
- resourcepacks/
- shaderpacks/
- README.txt
- modpack-manifest.json
- mod-list.md
- jvm-args.txt
- changelog.txt

推荐配置
- Java: 17
- 内存: 6GB 到 8GB
- 显卡: 独显 4GB 及以上体验更稳定

快速开始
1. 在启动器中新建 1.20.1 Fabric 实例。
2. 按 mod-list.md 下载依赖并放入 mods 目录。
3. 将 config、resourcepacks、shaderpacks 放入实例目录。
4. 把 jvm-args.txt 的参数填入启动器。
5. 首次进游戏后按需调整画质与光影。

注意
- 私人整合包请勿打包账号、日志、崩溃报告、缓存。
- 发布前请做一次干净环境导入测试。
