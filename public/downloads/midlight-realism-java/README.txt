Midlight Realism Java Pack

定位：中配设备可用的真实感增强资源包。

内容方向：
- 提高石材、木材、泥土和草地区块的真实感表现思路
- 配套轻量光影使用建议
- 优先兼顾 1080p 下的稳定性

建议环境：
- Minecraft Java Edition 1.20+
- Iris + Sodium 或 OptiFine

建议设置：
- 视距：12-20
- 阴影：中
- 反射：低或关闭
- 体积云：低或关闭

安装方式：
1. 将本压缩包放入 resourcepacks 文件夹
2. 启动游戏并在资源包中启用

说明：
本包为工具箱内置演示下载包，定位为中配视觉体验模板。
