下载与说明入口

以下跳转均为外部官方站点，新标签打开更稳妥。

入口 1

网易版官网

适合国服玩家，包含网易版客户端下载、版本活动和社区内容。

前往网易版下载 打开网易版官网
如果你平时使用网易启动器、手机国服或朋友都在网易版，一般从这里进入最合适。

入口 2

国际版官网

适合 Java 版、国际版基岩版玩家，也适合下载 Minecraft Launcher 和查看正版购买信息。

前往国际版下载 打开国际版官网
如果你玩 Java 正版、微软生态里的基岩版，或者需要 Minecraft Launcher，一般从这里进入。

选择建议

先确认你要去哪一边

玩网易国服：走网易版官网
玩 Java 正版：走国际版官网
要找 Minecraft Launcher：走国际版下载页
朋友都在国服联机：优先网易版
